# SANXZ Color Palettes (Source of Truth)

Este diretório armazena as definições de cores agnósticas em formato YAML.
Estes arquivos são a referência central para gerar configurações de temas.

## Autor
**Felipe Sanchez** (SRE/DevOps)

## Estrutura
Cada arquivo YAML contém:
1.  **Meta:** Informações do tema.
2.  **Palette:** Mapeamento ANSI 16 cores.
3.  **Semantic:** Cores funcionais (Erro, Aviso, Sucesso).
4.  **UI:** Cores de interface (Fundo, Texto).

## Temas Disponíveis
*   **`sanxz4.yaml`**: The Flagship. Swap White/Violet.
*   **`greenhill.yaml`**: Matrix Green Scale.
*   **`monored.yaml`**: Red/Purple Monochromatic.
*   **`warmcool.yaml`**: Warm Action, Cool Structure.
*   **`coolwarm.yaml`**: Cool Action, Warm Structure.
